# 조삼모사

### 문제 정보

- 평균 79분
- 55% 정답률
- 총 제출 2,025회

*n*개의 일이 주어질 때 이를 아침과 저녁으로 *n/2*개씩 나누어 처리하고자 합니다.

일마다 특성이 다르기 때문에 같이할 때의 업무 강도를 나타내는 업무 간의 상성 *P<sub>ij</sub>*가 존재합니다.

각 그룹의 총 업무 강도는

- 같은 그룹 내의 (i,j) 모든 쌍에 대해 P<sub>ij</sub> + P<sub>ji</sub> 의 합

아침과 저녁의 업무 강도의 차이의 최솟값을 구하는 프로그램을 작성하세요.

## 풀이법

- 코드 : **Java**

0. **일단 특정 상황에서 백트래킹이 불가하단 것을 깨닫는다.**

1. 재귀를 활용해 n/2 만큼의 조합을 만든다.
   1. index 순서대로 조합을 선택하고 시작 index 범위는 n/2 + 1까지 한정한다.
      (n/2 + 1번째 인덱스에서 시작해야 n-1까지 뽑았을 때 n/2개가 되기 때문에)
   2. Java였기 때문에 조합을 직접 구현했다. ㅎㅎ
2. 조합을 완성했을 때 그 나머지와 비교하여 차이의 최솟값을 갱신한다.
