# KB7-Algorithm-Study
KB It's your life 7기 알고리즘 스터디 레포지토리입니다.

- **일정**: 매주 수요일 (주 1회)
- **장소**: 미정
- **인원**:  **백광훈(스터디장)**, 권원영, 김충훈, 정민규, 황현진, 공서연, 김예림
- **스터디 방법**
    - 매주 각자의 난이도에 맞게 알고리즘 문제 3문제, 공통 알고리즘 문제 1문제(대기업 기출 문제)를 해결하여 공통 알고리즘 문제에 대한 서로의 사고방식 공유
- **스터디 목표**
    - 코딩테스트 능력 향상 및 다양한 대기업 문제 경향 파악
