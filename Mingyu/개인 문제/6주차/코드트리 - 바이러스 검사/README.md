## 📝 문제 설명

총 $n$개의 식당에 있는 고객들의 체온을 측정해야 한다.

검사자는 두 종류가 있다.

| 검사자   | 설명                                    |
| -------- | --------------------------------------- |
| 검사팀장 | 한 식당에 반드시 정확히 1명 있어야 한다 |
| 검사팀원 | 한 식당에 여러 명 배치할 수 있다        |

검사팀장은 한 명당 최대 $a$명의 고객을 검사할 수 있다.

검사팀원은 한 명당 최대 $b$명의 고객을 검사할 수 있다.

각 식당에는 검사팀장이 반드시 1명 있어야 하며, 검사팀원만 배치할 수는 없다.

또한 검사자는 자신이 배정된 식당의 고객만 검사할 수 있다.

목표는 모든 고객을 검사하기 위해 필요한 검사자 수의 최솟값을 구하는 것이다.

---

## 📥 입력

입력은 다음과 같은 형식으로 주어진다.

```text
n
c1 c2 ... cn
a b
```

| 변수  | 의미                                       |
| ----- | ------------------------------------------ |
| $n$   | 식당의 수                                  |
| $c_i$ | $i$번째 식당의 고객 수                     |
| $a$   | 검사팀장 1명이 검사할 수 있는 최대 고객 수 |
| $b$   | 검사팀원 1명이 검사할 수 있는 최대 고객 수 |

---

## 📤 출력

모든 식당의 고객을 검사하기 위해 필요한 검사자 수의 최솟값을 출력한다.

```text
최소 검사자 수
```

## 💡 핵심 아이디어

각 식당에는 검사팀장이 반드시 1명 필요하다.

따라서 먼저 모든 식당에 검사팀장을 1명씩 배치한다고 생각한다.

$$
answer = n
$$

$i$번째 식당의 고객 수를 $c_i$라고 하면, 검사팀장이 처리하고 남은 고객 수는 다음과 같다.

$$
rest = c_i - a
$$

남은 고객 수에 따라 경우를 나눈다.

| 조건         | 처리               |
| ------------ | ------------------ |
| $rest \le 0$ | 검사팀원 필요 없음 |
| $rest > 0$   | 검사팀원 추가 필요 |

검사팀원 1명은 최대 $b$명을 검사할 수 있다.

따라서 남은 고객이 있을 때 필요한 검사팀원 수는 다음과 같이 계산한다.

$$
ceil(rest / b)
$$

Java에서는 실수 연산 대신 정수 연산으로 올림 나눗셈을 처리한다.

$$
ceil(rest / b) = (rest + b - 1) / b
$$

## 🧮 정답 계산식

전체 정답은 다음 구조로 계산한다.

```text
answer = 검사팀장 수 + 검사팀원 수
```

검사팀장은 식당마다 1명씩 필요하다.

```text
검사팀장 수 = n
```

검사팀원 수는 각 식당에서 남은 고객 수에 따라 결정된다.

```text
각 식당마다:

rest = customer - leader

if rest <= 0:
    add = 0

if rest > 0:
    add = ceil(rest / member)
```

전체 정답은 다음처럼 생각할 수 있다.

```text
answer = n + sum of additional members
```

고객 수 기준으로 풀어 쓰면 다음과 같다.

```text
answer = n + sum of ceil((c_i - a) / b)
단, c_i - a > 0 인 경우에만 더한다.
```

## 🔍 구현 흐름

```text
1. 식당 수 n을 입력받는다.
2. 각 식당의 고객 수를 입력받는다.
3. 검사팀장의 검사 가능 수 leader를 입력받는다.
4. 검사팀원의 검사 가능 수 member를 입력받는다.
5. 각 식당마다 검사팀장 1명을 먼저 더한다.
6. rest = customer - leader 를 계산한다.
7. rest > 0 이면 검사팀원 수를 추가한다.
8. 최종 answer를 출력한다.
```

핵심 구현은 다음과 같다.

```java
answer++;

int rest = customer - leader;

if (rest > 0) {
    answer += (rest + member - 1) / member;
}
```

## ⚠️ 주의할 점

정답은 `int` 범위를 초과할 수 있다.

식당 수 $n$이 크고, 각 식당마다 필요한 검사팀원 수가 많다면 전체 검사자 수가 매우 커질 수 있다.

따라서 정답 변수는 반드시 `long` 타입을 사용해야 한다.

```java
long answer = 0;
```
