# 🧫 미생물 배양 실험 시뮬레이션 (C++)

## 📌 문제 개요

N×N 크기의 배양 용기에서 Q번의 실험을 진행하며,  
각 실험마다 미생물을 투입하고 결과 점수를 계산하는 시뮬레이션 문제이다.

---

## 🔄 실험 진행 순서

각 실험은 다음과 같은 순서로 진행된다.

1. 미생물 투입  
2. 쪼개진 미생물 제거  
3. 새 배양 용기로 이동  
4. 점수 계산  

---

## 🧠 핵심 아이디어

- **BFS** → 미생물이 하나의 덩어리인지 확인
- **정렬** → 큰 미생물부터 배치
- **상대좌표** → 모양 유지 이동
- **set** → 인접 쌍 중복 제거

---

## 🧪 1. 미생물 투입

직사각형 영역에 새로운 미생물을 덮어쓴다.

```cpp
for (int r = r1; r < r2; r++) {
    for (int c = c1; c < c2; c++) {
        board[r][c] = id;
    }
}
```
---

📌 **주의**  
좌표는 `[r1, r2)`, `[c1, c2)` (끝점 미포함)

---

## 💥 2. 쪼개진 미생물 제거

새로운 미생물이 기존 미생물을 일부 먹었을 때,  
기존 미생물이 여러 덩어리로 나뉘면 전체 삭제된다.

### ✔ BFS로 연결성 확인

```cpp
bool isConnected(int id)
```
### 🔍 동작 과정

- 해당 id의 모든 좌표를 수집
- 하나의 좌표에서 BFS 시작
- 방문 가능한 칸 개수 확인
- BFS 방문 개수 == 전체 칸 개수 → 연결됨
- BFS 방문 개수 < 전체 칸 개수 → 쪼개짐 → 삭제

## 🚚 3. 새 배양 용기로 이동

모든 미생물을 새로운 배양 용기로 옮긴다.

### 📌 이동 규칙
- 넓이가 큰 미생물부터 이동
- 넓이가 같으면 먼저 투입된 미생물 (id 작은 것)
- 모양 유지 (회전 ❌)
- 겹치면 안 됨
- 가능한 위치 중
- 행(row)이 가장 작은 위치
- 같으면 열(column)이 가장 작은 위치
- 놓을 수 없으면 사라짐

### 정렬 기준
```
sort(groups.begin(), groups.end(), [](Group& a, Group& b) {
    if (a.area != b.area) return a.area > b.area;
    return a.id < b.id;
});
```

### 🧠 상대좌표 변환
(5,7), (5,8), (6,7)
->(0,0), (0,1), (1,0)

👉 위치는 바뀌지만 모양 유지

---


## 🔗 4. 점수 계산

### 📌 규칙
서로 다른 미생물이 맞닿아 있으면
(넓이 A × 넓이 B) 만큼 점수 추가
같은 쌍은 한 번만 계산

### 🔧 구현 핵심
```
set<pair<int, int>> pairs;
if (a > b) swap(a, b);
pairs.insert({a, b});
```

### 📦 사용 자료구조
- vector<pair<int,int>> → 좌표 저장
- queue<pair<int,int>> → BFS 탐색
- set<int> → 미생물 id 중복 제거
- map<int, vector<pair<int,int>>> → id별 좌표 그룹화
- set<pair<int,int>> → 인접 쌍 중복 제거
