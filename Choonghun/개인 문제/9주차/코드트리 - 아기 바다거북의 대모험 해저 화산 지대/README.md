<h3 id="턴-진행"><strong>턴 진행</strong></h3>
<h4 id="1단계-바다거북-이동"><strong>1단계: 바다거북 이동</strong></h4>
<p>바다거북은 ID가 작은 순서대로(1번부터 <span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>M</mi></mrow><annotation encoding="application/x-tex">M</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height: 0.6833em;"></span><span class="mord mathnormal" style="margin-right: 0.10903em;">M</span></span></span></span>번까지) 한 마리씩 이동합니다. 각 거북이는 자신의 차례에 현재 바다 상태를 기준으로 안식처까지의 <strong>최단 경로</strong>를 탐색합니다.</p>
<ul>
<li><strong>이동 방해 요소:</strong> <strong>산호초(1)</strong>, 다른 살아있는 바다거북, 그리고 열기에 지쳐 굳어버린 <strong>화석</strong>은 모두 통과할 수 없는 장애물로 간주합니다.</li>
<li><strong>이동 규칙:</strong> 최단 경로가 존재한다면 해당 경로의 첫 번째 칸으로 한 칸 이동합니다.
<ul>
<li>최단 경로가 여러 개라면 <strong>우(→), 하(↓), 좌(←), 상(↑)</strong> 순서로 우선순위를 두어 첫 이동 방향을 결정합니다.</li>
<li>최단 경로가 존재하지 않는다면 이동하지 않고 제자리에 머뭅니다.</li>
</ul>
</li>
<li><strong>안식처 도착:</strong> 거북이가 <strong>안식처(<span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mo stretchy="false">(</mo><mi>N</mi><mo>−</mo><mn>1</mn><mo separator="true">,</mo><mi>N</mi><mo>−</mo><mn>1</mn><mo stretchy="false">)</mo></mrow><annotation encoding="application/x-tex">(N-1, N-1)</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height: 1em; vertical-align: -0.25em;"></span><span class="mopen">(</span><span class="mord mathnormal" style="margin-right: 0.10903em;">N</span><span class="mspace" style="margin-right: 0.2222em;"></span><span class="mbin">−</span><span class="mspace" style="margin-right: 0.2222em;"></span></span><span class="base"><span class="strut" style="height: 0.8778em; vertical-align: -0.1944em;"></span><span class="mord">1</span><span class="mpunct">,</span><span class="mspace" style="margin-right: 0.1667em;"></span><span class="mord mathnormal" style="margin-right: 0.10903em;">N</span><span class="mspace" style="margin-right: 0.2222em;"></span><span class="mbin">−</span><span class="mspace" style="margin-right: 0.2222em;"></span></span><span class="base"><span class="strut" style="height: 1em; vertical-align: -0.25em;"></span><span class="mord">1</span><span class="mclose">)</span></span></span></span>)</strong> 에 도착하면 즉시 지도에서 제외되며, 해당 턴을 도착 시간으로 기록합니다.</li>
<li><strong>참고:</strong> 거북이는 해저 화산이 있는 칸으로 진입할 수 있습니다. 또한, 앞선 거북이의 이동 결과는 즉시 다음 거북이의 경로 탐색에 반영됩니다.</li>
</ul>
<h4 id="2단계-화산-압력-증가"><strong>2단계: 화산 압력 증가</strong></h4>
<p>바다에 존재하는 모든 해저 화산의 마그마 압력이 각각 <strong>10씩 증가</strong>합니다.</p>
<h4 id="3단계-화산-분출-및-연쇄-반응"><strong>3단계: 화산 분출 및 연쇄 반응</strong></h4>
<p>현재 마그마 압력이 각 화산의 <strong>분출 임계치(<span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>P</mi></mrow><annotation encoding="application/x-tex">P</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height: 0.6833em;"></span><span class="mord mathnormal" style="margin-right: 0.13889em;">P</span></span></span></span>) 이상</strong>인 화산은 뜨거운 열기를 분출합니다.</p>
<ol>
<li><strong>열기 전파:</strong>
<ul>
<li>화산이 위치한 칸에 해당 화산의 <strong>분출 임계치(<span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>P</mi></mrow><annotation encoding="application/x-tex">P</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height: 0.6833em;"></span><span class="mord mathnormal" style="margin-right: 0.13889em;">P</span></span></span></span>)</strong> 만큼의 열기가 발생합니다.</li>
<li>열기는 상·하·좌·우 4방향으로 뻗어 나가며, 한 칸 이동할 때마다 이전 칸 열기의 절반(<span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mo stretchy="false">⌊</mo><mtext>열기</mtext><mi mathvariant="normal">/</mi><mn>2</mn><mo stretchy="false">⌋</mo></mrow><annotation encoding="application/x-tex">\lfloor 열기 / 2 \rfloor</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height: 1em; vertical-align: -0.25em;"></span><span class="mopen">⌊</span><span class="mord hangul_fallback">열기</span><span class="mord">/2</span><span class="mclose">⌋</span></span></span></span>, 소수점 내림)이 됩니다.</li>
<li><strong>산호초</strong>를 만나거나 열기 값이 0이 되면 해당 방향의 전파는 중단됩니다.</li>
<li>한 칸에 여러 화산의 열기가 도달하면 그 값들을 모두 합산합니다.</li>
</ul>
</li>
<li><strong>연쇄 반응:</strong>
<ul>
<li>아직 분출하지 않은 화산 중, <strong>(현재 마그마 압력 + 해당 칸에 누적된 외부 열기) <span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mo>≥</mo></mrow><annotation encoding="application/x-tex">\ge</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height: 0.7719em; vertical-align: -0.136em;"></span><span class="mrel">≥</span></span></span></span> 분출 임계치(<span class="katex"><span class="katex-mathml"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>P</mi></mrow><annotation encoding="application/x-tex">P</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height: 0.6833em;"></span><span class="mord mathnormal" style="margin-right: 0.13889em;">P</span></span></span></span>)</strong> 가 되면 해당 화산도 즉시 분출을 시작합니다.</li>
<li>외부 열기는 분출 조건만 만족시킬 뿐, 해당 화산의 실제 마그마 압력 수치 자체를 증가시키지는 않습니다.</li>
<li>새로 분출하는 화산이 없을 때까지 이 연쇄 분출 과정은 반복됩니다.</li>
</ul>
</li>
<li><strong>바다거북의 위기(화석화):</strong>
<ul>
<li>모든 분출이 종료된 후, 살아있는 거북이가 위치한 칸의 <strong>총 열기 합이 20 이상</strong>이면 해당 거북이는 <strong>화석</strong>이 됩니다.</li>
<li>화석이 된 거북이는 그 자리에 고정되어, 이후 턴부터 다른 거북이들의 이동을 방해하는 장애물이 됩니다.</li>
</ul>
</li>
</ol>
