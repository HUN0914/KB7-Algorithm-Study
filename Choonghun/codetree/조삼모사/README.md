## 📝 문제 설명
n개의 일을 두 그룹(아침, 저녁)으로 나누어 각각 n/2개씩 배치합니다.

각 일 i, j 사이에는 함께 수행할 때의 업무 강도 P<sub>ij</sub>가 존재합니다.

각 그룹의 총 업무 강도는 다음과 같이 계산됩니다.

- 같은 그룹 내 모든 쌍 (i, j)에 대해 P<sub>ij</sub> + P<sub>ji</sub>의 합

두 그룹(아침, 저녁)의 업무 강도 차이를 최소로 만드는 값을 구해야 합니다.
